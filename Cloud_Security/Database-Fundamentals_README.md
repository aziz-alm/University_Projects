# 31271 Database Fundamentals — UTS

**Final grade: 100/100 (High Distinction)**

Core UTS IT subject on the fundamentals of relational database systems: how data is structured and managed so it can be used effectively by applications and users, and how to use SQL to retrieve and modify that data. The work runs the full design lifecycle — from a written case study through to a working PostgreSQL database.

## Projects

| Project | Description | Result |
|---|---|---|
| [Apple App Store — Relational Database Design](./apple-app-store-db-design) | End-to-end database design and implementation, delivered across four parts: case study → ERD → normalisation → SQL build. | 100/100 |

## Concepts covered
- Conceptual database design and ER modelling
- Logical design — converting an ERD to a relational model
- Functional dependencies and normalisation
- SQL — queries, aggregate functions, joins, subqueries, and set operators

## Tools
PostgreSQL · SQL · ER modelling
