# Digital Forensics (48436)

**Grade: High Distinction**

Practice-based UTS subject covering the acquisition, analysis, and reporting of digital evidence — network forensics, disk imaging, FAT32/NTFS file systems, Windows Registry analysis, Linux artifacts, and metadata examination.

## Contents

### [Skills Based Report](./Skills%20Based%20Report)
Forensic investigation of a disk image acquired from a suspect's laptop. The task: determine whether allegations of misusing company time and intellectual property could be proven. Covers evidence acquisition, artifact analysis, and a defensible written report following sound forensic principles.

## Key skills
- Forensically sound evidence handling and chain of custody
- Disk and network data acquisition
- Windows Registry and file system analysis
- Metadata analysis (documents, graphics, camera images)
